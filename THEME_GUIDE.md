# 主题切换功能说明

## 🎨 功能概述

现已为SeismicX地震监测系统添加了完整的主题切换功能，支持在夜间模式和日间模式之间无缝切换。

## 🌙 夜间模式 (Dark Theme)
- **背景色**: 深黑色 (#0a0a0a)
- **卡片色**: 深灰色 (#141414)  
- **文字色**: 白色 (#ffffff)
- **强调色**: 橙色 (#f97316)
- **边框色**: 深灰色 (#262626)
- **网格背景**: 深色点阵图案

## ☀️ 日间模式 (Light Theme)
- **背景色**: 浅灰白色 (#f8fafc)
- **卡片色**: 白色 (#ffffff)
- **文字色**: 深蓝色 (#1e293b)
- **强调色**: 深橙色 (#ea580c)
- **边框色**: 浅灰色 (#e2e8f0)
- **网格背景**: 浅色点阵图案

## 🎛 使用方法

### 1. 快速切换
- 点击右上角的太阳/月亮图标按钮
- 单击即可在两个主题间切换

### 2. 详细选择
- 点击主题按钮后会出现下拉选择器
- 可以清楚地看到两个主题的预览
- 点击对应的主题卡片进行选择

### 3. 自动保存
- 主题选择会自动保存到浏览器本地存储
- 下次访问时会记住上次的选择
- 支持跨会话的主题偏好设置

## 🛠 技术实现

### 架构设计
```
src/
├── context/
│   └── ThemeContext.tsx      # 主题状态管理
├── components/
│   └── ThemeToggle.tsx       # 主题切换UI组件
└── index.css                 # CSS主题变量系统
```

### 核心特性
- **React Context**: 全局主题状态管理
- **CSS变量**: 动态主题切换
- **平滑过渡**: 300ms渐变动画
- **本地存储**: 用户偏好持久化
- **响应式设计**: 适配不同屏幕尺寸

### 主题变量
```css
:root[data-theme="dark"] {
  --color-bg: #0a0a0a;
  --color-card: #141414;
  --color-accent: #f97316;
  --color-border: #262626;
  --color-text: #ffffff;
  --color-text-muted: #737373;
}

:root[data-theme="light"] {
  --color-bg: #f8fafc;
  --color-card: #ffffff;
  --color-accent: #ea580c;
  --color-border: #e2e8f0;
  --color-text: #1e293b;
  --color-text-muted: #64748b;
}
```

## 🔧 开发者说明

### 扩展新主题
1. 在CSS中添加新的`:root[data-theme="xxx"]`规则
2. 在ThemeSelector组件中添加新主题选项
3. 更新Theme类型定义

### 自定义主题变量
可以在CSS变量中添加任何需要的主题属性：
```css
--color-success: #10b981;
--color-warning: #f59e0b;
--color-error: #ef4444;
--color-info: #3b82f6;
```
